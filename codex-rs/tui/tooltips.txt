对话过长时使用 /compact 压缩历史记录，释放上下文空间。
使用 /new 开始新话题；之前的会话仍保留在历史记录中。
使用 /feedback 向维护者发送日志，反馈问题。
使用 /model 快速切换模型或推理强度。
使用 /permissions 控制 Codex 何时需要确认。
使用 /review 对当前变更进行代码审查。
使用 /skills 查看可用技能，或让 Codex 使用某个技能。
使用 /status 查看当前模型、审批设置和 token 用量。
使用 /statusline 配置状态栏中显示的项目。
使用 /fork 将当前对话分叉到新线程。
使用 /init 创建 AGENTS.md，添加项目专属指引。
使用 /mcp 查看已配置的 MCP 工具。
运行 `codex app` 打开 Codex 桌面版（macOS 上如未安装会自动安装）。
使用 /personality 自定义 Codex 的交流风格。
使用 /rename 重命名线程，方便后续恢复会话。
使用 OpenAI 文档 MCP 回答 API 问题；通过 `codex mcp add openaiDeveloperDocs --url https://developers.openai.com/mcp` 启用。
加入 OpenAI 社区 Discord：http://discord.gg/openai
访问 Codex 社区论坛：https://community.openai.com/c/codex/37
在 Codex 中可以使用 `!` 运行任意 shell 命令（如 `!ls`）
输入 / 打开命令面板；Tab 自动补全斜杠命令。
输入框为空时，按 Esc 回退编辑上一条消息；Enter 确认。
任务运行时按 Tab 排队消息；否则立即发送（`!` 除外）。
使用 Ctrl+V 粘贴图片，附加到下一条消息。
运行 `codex resume` 可恢复之前的对话
